# Topic Modelling from Unstructured Text

This project applies unsupervised NLP techniques to discover latent themes
within unstructured textual data using statistical topic modelling methods.


## Objective

- Extract meaningful topics from unstructured text
- Compare probabilistic and matrix-factorisation approaches
- Visualise and interpret topic distributions

## Techniques Used

### Preprocessing
- Tokenisation
- Stopword removal
- Lemmatization

### Vectorisation
- CountVectorizer
- TF-IDF

### Topic Models
- Latent Dirichlet Allocation (LDA)
- Non-negative Matrix Factorisation (NMF)

## Visualisations

- Word clouds
- Topic-term heatmaps
- PCA-based topic clustering

## How to Run

1. Place dataset in the `dataset/` folder
